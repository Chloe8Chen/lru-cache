# lru-cache
lru-cache is a JavaScript project that implements a LRU (Least Recenly Used) Cache

## Setup

Clone the repo from https://github.com/Chloe8Chen/lru-cache.git


## Run

- To run the example:
```node examples.js```

- To run the unit tests:
```npm test```

- To run the coverage:
```npm run coverage```
